# 术语黑名单 (Buzzword Blacklist)
# 作用：检测文档中的"空话"和"虚词"，用于判定文档的清晰度和务实程度。

# 格式：关键词 | 权重(1-5) | 建议替换词/说明
# 权重越高，代表该词越空洞，扣分越重。
